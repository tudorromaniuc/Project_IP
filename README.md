Proiectul meu are legatura cu derivarea de functii
Se dă o funcție (sub forma unui șir de caractere) și se cere să se obțină derivata sa (prima derivata si a doua derivata) simbolică (formală).
Implementari:
- Se poate transforma expresia funcției în forma postfixată
- Se creează arborele binar asociat expresiei
- Se obține din acesta arborele derivatei
- Se generează noua expresie, dupa câteva simplificări ale arborelui derivat.
